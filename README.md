# CS 330 Computational Graphics and Visualization

## Overview
This repository contains work completed for CS 330, Computational Graphics and Visualization. In this course, I used C++ and OpenGL to build and modify 3D scenes while applying graphics concepts such as transformations, lighting, textures, camera movement, and multiple viewing modes.

## Course Focus
The main goal of this course was to develop interactive 3D scenes and strengthen my understanding of how computer graphics are built and controlled in code. The projects in this repository demonstrate work with object placement, scaling, rotation, lighting effects, textures, and scene navigation.

## Project Highlights
- Built and modified 3D scenes using OpenGL
- Applied transformations such as translation, rotation, and scale
- Used ambient, diffuse, and specular lighting
- Worked with textures to improve object appearance
- Used perspective and orthographic views
- Adjusted camera movement and scene navigation
- Worked within a modular codebase that separated scene and view behavior

## Tools and Technologies
- C++
- OpenGL

## What I Learned
Through this course, I gained experience building 3D objects from basic shapes and controlling how they appear in a scene. I also learned how lighting and textures affect realism, and how camera controls change the user experience. This project helped me better understand how graphics programming combines math, design, and programming structure.
